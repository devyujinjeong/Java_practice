## 📜 **커밋 메시지 규칙**
실습 기록을 명확하게 관리하기 위해 다음과 같은 커밋 메시지 규칙을 따릅니다. (실습 10부터 적용됨)

- ✨ `feat`: 새로운 실습 코드 추가
- 📝 `docs`: README 또는 문서 수정
- 🛠 `refactor`: 코드 리팩토링
- 🚀 `chore`: 환경 설정 및 기타 변경
- 🐞 `fix`: 버그 수정
